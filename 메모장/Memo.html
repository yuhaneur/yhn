1. 여백에는 종류가 2개 태그자체 여백키우는건 패딩 다른태그와 간격을 벌려주는건 마진
    ul태그는 안쪽과 바깥족 여백이있음
  2. ol 과 ul 은 li태그를 사용 (li태그만 사용하면안됨)
  3. dl은 책목차(소제목과 내용)같은 목록을 나타낼때 사용 dt와 dd를 사용
  4. li*3하면 3개가 한번에 만들어짐
  5. ul>li*2하면 ul태그하위에 li태그 2개만들어라 > 의 의미는 하위태그를 의미함
    ul>li{목록3}*2

자주사용하는 태그들

  • ul - 순서를 표시하지않는 리스트 태그
    p- 단락을 표현하는 태그이지만, 문장의 형식 글을 표현할때 많이 사용한다.
    b- 텍스트에 굵기를 굵게 해주는 태그 굵게 표시하기위한 목적보다는 단어 또는 단편적인 텍스트 표현할때 사용된다.
    h1,h2,h3,~6 - 페이지에서 컨텐츠들의 제목이나 메인 글을 표현 할때 사용된다.
